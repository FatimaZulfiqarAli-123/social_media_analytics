# 📊 Social Media Analytics Dashboard

This project analyzes social media performance across platforms like Facebook, Instagram, and LinkedIn using Python and Streamlit.

---
## 📊 Features

✔ Real-time social media data analysis  
✔ Engagement calculation (Likes + Comments + Shares)  
✔ Interactive Streamlit dashboard  
✔ Platform-wise comparison  
✔ Top performing posts visualization  

---

## 🛠 Tech Stack

- Python 🐍
- Pandas 📊
- Streamlit ⚡
